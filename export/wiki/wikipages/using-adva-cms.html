Using adva_cms

Start server by using the DOS window PC or Terminal (MAC). Enter 
cd adva_cms (or other folder name)
for mac: 
script/server
for windows: 
ruby script/server
Now you can use adva cms in your browser window using the address: 
localhost:3000/admin
